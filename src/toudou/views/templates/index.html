Welcome to the toudou project

Press the + icon to beggin your toudou journey

1. Add a todo | 2. Edit a todo | 3. Delete a todo | 4. Download todos

1. How to add a todo

First go on the add page by clicking on the menu.

Scroll dow until you see the create form

Fill up the required fields and click on submit card

2. How to edit a todo

First go on the view page by clicking on the menu.

Then click on the pencil icon on the card you want to edit,

Scroll dow until you see the edit form

Fill up the required fields and click on submit changes

3. How to delete a todo

First go on the view page by clicking on the menu.

Then click on the bin icon on the card you want to delete,

a confirmation box will appear to confirm if you want to delete this task,

click ok to delete the task FOREVER

4. How to download todos

First go on the view page by clicking on the menu.

Then click on the download icon,

the download will start automatically
